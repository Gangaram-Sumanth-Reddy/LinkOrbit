🖼️ Header Background Examples

Interactive examples for setting custom header backgrounds

1. Set Background from URL

Use a direct image URL to set the header background. Works with local files or external URLs.

setHeaderBackground('asserts/header-bg.jpg'); // or setHeaderBackground('https://images.unsplash.com/photo-1557683316-973673baf926?w=800');

2. Upload Custom Image

Upload your own image file to use as the header background.

const fileInput = document.getElementById('fileInput'); fileInput.addEventListener('change', (e) => { uploadHeaderBackground(e.target.files[0]); });

3. Remove Background

Remove the custom background and restore the default gradient.

removeHeaderBackground();

4. Automatic Slideshow

Rotate through multiple images automatically.

const images = ['img1.jpg', 'img2.jpg', 'img3.jpg']; let index = 0; setInterval(() => { setHeaderBackground(images[index]); index = (index + 1) % images.length; }, 5000);

5. Random Background

Set a random image from a collection.

const images = ['img1.jpg', 'img2.jpg', 'img3.jpg']; const random = images[Math.floor(Math.random() * images.length)]; setHeaderBackground(random);
📝 Note
These examples will work when you open your main index.html file. Make sure you have the header background functions loaded in script.js.
